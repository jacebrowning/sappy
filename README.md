## Getting Started

### Requirements

* Python 2.7+ or Python 3.3+

### Installation

sappy can be installed with pip:

```
$ pip install sappy
```

or directly from the source code:

```
$ git clone https://github.com/jacebrowning/sappy.git
$ cd sappy
$ python setup.py install
```
